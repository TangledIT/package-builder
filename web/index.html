Repositories

  • Tangled IT provides APT and Yum repositories for installing Tangle (IOTA) based software.

    The packages require Systemd init system, which is now the default for most modern linux based operating systems.

  • Getting started

    We've tried to adhere to the standards of the linux OS and it's directory structures. This means the config is stored in /etc/package-name and additional files in /var/lib/package-name. If you would like to check if Systemd is installed, you can run: systemctl status

  • Packages

    You can install the following packages:

    * iri - The IOTA Reference Implementation (source) (package info)

    * ict - Iota Controlled AgenT (source) (package info)

    * report-ixi - Report IXI, a IXI plugin to report node information (source) (package info)

    * nelson - Nelson is a IRI neighbor discovery service created by SemkoDev (source) (package info)

    * redix - A persistent key-value store based on RESP (Redis) protocol by alash3al (source) (package info)


    If you want to build the packages on your own, you can do that, the package-builder is open-source and can be found here:

    package-builder

  • APT (Debian / Ubuntu)

    The first thing you'll need to do is add our repository and install the public key. curl -sSL https://repos.tangled.it/apt/setup.sh | sudo bash This will execute the following code. If you prefer to do it manually you can run these lines: apt-key adv --keyserver hkp://keyserver.ubuntu.com:80 --recv-keys EA0CD139C90637FB echo "deb https://repos.tangled.it/apt/ tangledit main" > /etc/apt/sources.list.d/tangledit.list apt-get update After the repository has been added you can install packages from it. For example: apt-get install iri

  • Yum (Redhat / CentOS)

    The first thing you'll need to do is add our repository and install the public key. curl https://repos.tangled.it/yum/setup.sh | sudo bash This will execute the following code. If you prefer to do it manually you can run these lines: rpm --import https://repos.tangled.it/repo@tangled.it.gpg.key cat > /etc/yum.repos.d/tangledit.repo <<END [tangledit] name = TangledIT baseurl = https://repos.tangled.it/yum/ END yum -y update After the repository has been added you can install packages from it. For example: yum install iri
